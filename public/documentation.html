Movie API

Objective: A REST API serving requests about movie data. The API will handle client requests for specific movie data, genre collections, film directors, including their latest releases and their filmographies. The Client will also be able to generate personal accounts where they can create and edit lists of favorite films and directors and also have access to a concise account deletion flow.

Movie API request-response cycle

Method Endpoint API Function Request Body Response Body
GET /movies return complete list of movies none JSON array of all movies
GET /movies/:title return single movie's data none JSON data object of 1 movie filtered by title parameter:
  • "title":"Batman"
  • "director":"Matt Reeves
  • "releaseYear":"2022"
  • "studio":"2022"
  • "synopsis":"Caped crusader solves crimes in gotham"
  • "genre": { "name": "action", "description": "high stakes, special effects, setpieces" }
  • "studio":"Warner Bros"
  • "country":"us"
GET /directors return complete list of directors none JSON array of all directors
GET /directors/:name return single director's data none JSON data object of 1 director filtered by name parameter:
  • "name": "David Lowery"
  • "latestFilm: "The Green Knight"
  • "bio": "experimental film director, whose new filmography already spans a large range of genres"
  • "country": "US"
GET /genres return complete list of genres none JSON array of all genres
GET /genres/:type return single genre's data none JSON data object descrbing 1 genre filtered by type parameter
POST /users register a new user account JSON object containing data for new account to add to database:
  • "name" : "Jill Smith"
  • "username" : "jillLovesMovies"

 JSON object containing the data of the added account and a success message:

  • "username" : "jillLovesMovies"
  • "email" : "jill@gmail.com"

  • "username:" "new user name"

  • "username:" "new user name"
